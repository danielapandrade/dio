Tags h1 a h6

São tags de título e subtítulos, partindo da maior preferência h1 até a menor h6.

Tags p

São tags de parágrafos

Tag mark

É uma tag de marcar o texto, semelhante a um marcador.

Tag small

Define um texto de tamanho menor.

Tag i

Deixa o texto em italico

Tag u

Deixa o texto sublinhado

Tag Strong

Tag usada para marcar um conteúdo como de grande importância, dando destaque de negrito ao texto

Tag ol

Tag que indica lista ordenada

Tag ul

Tag que indica lista não ordenada

Tag li

Indicam os itens da lista

Tag a

Tag de ancoragem, que possibilita ligar uma página/elemento a outro

Tag hr

Tag que dá uma linha horizontal no documento

Tag sub

Deixa o texto subscrito

Tag sup

Deixa o texto sobrescrito

Blockquote

é uma tag para citação

Tag font

Foi usada no HTML 4 para dar formatação ao texto, como tamanho, cor, foi descontinuada no HTML 5.

Tag del

Representa texto excluído do documento

Tag abbr

Tag de acrônimo, HTML: Hypertext Markup Language
